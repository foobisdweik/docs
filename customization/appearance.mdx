---
title: "Customize Spacedrive's interface layout and display"
description: "Adjust sidebar sections, thumbnail sizes, list density, font scale, accent color, and vibrancy in Spacedrive's Appearance settings panel."
---

Everything that affects how Spacedrive looks and feels — sidebar visibility, file grid density, text size, and background effects — lives in one place: **Spacedrive → Settings → Appearance**. Changes apply immediately without restarting the app.

## Opening Appearance settings

Choose **Spacedrive → Settings** from the menu bar, or press **⌘,**, then select **Appearance** in the left sidebar.

## Sidebar

The sidebar shows your Locations, Volumes, and Cloud sections by default. You can hide any section you rarely use:

- Toggle **Locations**, **Volumes**, or **Cloud** visibility using the checkboxes under **Sidebar → Sections**.
- Drag the sidebar divider to adjust its width, or enter an exact pixel value in the **Sidebar width** field.

## Grid view — thumbnail size

When browsing in grid view, the thumbnail size slider lets you choose how large each file tile appears. The four preset stops are **Small**, **Medium**, **Large**, and **Extra-large**. Larger thumbnails make it easier to visually identify files; smaller thumbnails fit more files on screen at once.

## List view — density

In list view, choose between two row heights:

- **Compact** — tighter row spacing, more files visible without scrolling.
- **Comfortable** — slightly more padding per row, easier to read at a glance.

## Font scale

The **Font scale** setting adjusts the size of all text in the interface without changing the operating system font size. Available options are **Small**, **Medium** (default), and **Large**.

## Accent color

The accent color controls the highlight used for selected files, focused inputs, the active sidebar item, and interactive controls. Click the color swatch next to **Accent Color** to open the picker and choose any color. To reset to the default blue (`#2499FF`), click **Reset to default** below the picker.

<Tip>
For a more detailed look at the full theme system — including the upcoming theme pack feature — see [Switch and customize Spacedrive themes](/customization/themes).
</Tip>

## macOS vibrancy

When **Vibrancy** is enabled, the sidebar background becomes translucent and blurs the content behind it, matching the macOS Tahoe system appearance. This setting follows the system-level transparency preference: if you have turned off transparency in **System Settings → Accessibility → Display**, Spacedrive will use a solid background regardless of this setting.

## Window corners

Spacedrive's window uses rounded corners that match macOS Tahoe system chrome. This is applied automatically and does not have a toggle — it reflects the system window geometry for consistency with other native apps on macOS Tahoe.
