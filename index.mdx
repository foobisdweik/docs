Spacedrive is a macOS Tahoe (26.5+) file manager built to replace Finder as your daily driver. It handles local disks, external drives, and cloud buckets — S3, Google Drive, OneDrive, Dropbox, WebDAV — through a single unified interface. All your data stays local: no accounts, no telemetry, no sync fees.

## Get up and running

<Steps>
  <Step title="Install Spacedrive">
    Download the `.dmg` from the releases page, open it, and drag **Spacedrive.app** to your Applications folder. macOS Tahoe 26.5 or later is required.
  </Step>
  <Step title="Grant permissions">
    On first launch, Spacedrive will ask for **Full Disk Access**. Open **System Settings → Privacy & Security → Full Disk Access** and enable Spacedrive. Without this, some folders will not be indexed.
  </Step>
  <Step title="Create a library">
    A library is where Spacedrive stores its index. Click **New Library** in the sidebar, give it a name, and choose a folder to save it in.
  </Step>
  <Step title="Add locations">
    Add local folders or cloud buckets to your library. Spacedrive indexes them in the background and generates thumbnails automatically.
  </Step>
</Steps>

## Why Spacedrive?

<CardGroup cols={2}>
  <Card title="Local-first" icon="hard-drive">
    Your index lives in an SQLite database on your Mac. No cloud account needed to use any core feature.
  </Card>
  <Card title="Cloud-aware" icon="cloud-arrow-up">
    Auto-detects iCloud, Google Drive, OneDrive, and Dropbox mounts. Connect any S3-compatible bucket with a key pair.
  </Card>
  <Card title="Finder integration" icon="apple">
    Deep macOS integration: Finder Sync overlays, Quick Look previews, Core Spotlight search, Services menu, and default folder handler.
  </Card>
  <Card title="GPU thumbnails" icon="image">
    VideoToolbox and Metal-accelerated thumbnail pipeline. HEIC, ProRes, H.264, HEVC — all rendered fast, with a software fallback.
  </Card>
